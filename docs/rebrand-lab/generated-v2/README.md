## Prompt set

Every direction asked for a high-fidelity 16:10 review board with a dominant
macOS inbox/open-message state, an iPhone inbox, realistic mail labels, integrated
native title bars and tabs, compact mobile navigation, accessible contrast, and
no controls beneath a tab bar. The icon was specified as the brand/material
reference and the live UI as the information-architecture reference.

Direction-specific prompt briefs:

1. **Tactile Atelier** — premium stationery, bookbinding, machined brass, barely
   perceptible grain, forest/bone/brass/coral palette, light primary plus dark inset.
2. **Editorial Correspondence** — independent print journal, asymmetric hierarchy,
   fine rules, almost no cards, narrow forest rail, vermilion active mark.
3. **Quiet Native Utility** — contemporary macOS/iOS split views and toolbars,
   crisp neutral surfaces, signature green account rail, minimal theming.
4. **Postal Grid** — Swiss postal modernism, strong grid, stamped status marks,
   squared controls, off-white/ink/green/ochre/coral palette.
5. **Warm Paper Desk** — restrained folded-paper tactility, archival folders,
   shallow elevation, modern interactions, credible night-desk counterpart.
6. **Nocturne Brass** — dark-mode-first editorial utility, charcoal and warm panels,
   moss account rail, pinpoint brass actions, coral unread markers, light inset.

Shared negative constraints: no generic SaaS dashboard, purple gradients,
glassmorphism, oversized cards or headings, rounded-rectangle soup, illegible tiny
text, decorative clutter, controls beneath tab bars, random gibberish, or watermark.
