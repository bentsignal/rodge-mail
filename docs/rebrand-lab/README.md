# Rodge Mail rebrand lab

This lab explores four product-level directions without changing the shipping
web, desktop, or mobile shells. Open `index.html` in a browser and use the
direction, appearance, and viewport controls to compare them.

These are deliberately different interaction models, not palettes:

- **Dispatch** — a dense, keyboard-first operations console with a persistent
  account dock and compact triage queue.
- **Correspondence** — an editorial reading room where the inbox is a table of
  contents and messages open as generous pages.
- **Frequency** — a dark-first, high-contrast communications instrument with a
  temporal pulse and a focus-first queue.
- **Orbit** — a spatial, touch-forward workspace with account orbits, stacked
  message cards, and a reader sheet.

Every direction includes a designed light and dark mode plus a responsive
mobile composition. The current production shell remains the fallback until a
direction is selected.
